Sponsored Resources

Atmel will be providing you with XPlained MINI development boards featuring the ATMega328P microcontroller, used on the very popular Arduino Uno board. You will be able to program with the Arduino Shield Tools and then use Atmel’s Studio 6.2 for programming and debugging.

In order to get your PC ready to start programming on Saturday, we suggest that you complete the following 3 steps:

Download the Arduino Tools
Download the Atmel Studio 6.2 Tools
Note: you will be asked to register for “My Atmel” before you continue with the downloading process
Install the Arduino Extensions for Atmel Studio 6.2 (Tools > Extensions Manager... > Search for Arduino IDE for Atmel Studio)
